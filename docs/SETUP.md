# Настройка кошелька для Polycashmashine

Пошаговая инструкция: от нуля до работающего бота.

---

## Шаг 1: Создать аккаунт Polymarket

1. Открой **https://polymarket.com**
2. Нажми **Sign Up**
3. Выбери способ входа:
   - **Email** (рекомендую — проще всего для бота) → `SIG_TYPE = 1`
   - **MetaMask / Coinbase Wallet** → `SIG_TYPE = 0`
4. Подтверди email или подключи кошелёк

---

## Шаг 2: Пополнить баланс USDC

На Polymarket ставки в **USDC** (стейблкоин, 1 USDC = $1) на сети **Polygon**.

### Вариант A: Карта/банк (самый простой)
1. В Polymarket нажми **Deposit**
2. Выбери **Credit/Debit Card** или **Bank Transfer**
3. Внеси $50–100 для старта (минимум ~$20)
4. USDC автоматически появится на балансе

### Вариант B: Крипто-перевод
1. В Polymarket нажми **Deposit** → **Crypto**
2. Скопируй свой **Polygon USDC адрес**
3. Отправь USDC с биржи (Binance, Bybit и т.д.) на этот адрес
4. Сеть: **Polygon** (не Ethereum — иначе потеряешь деньги)

---

## Шаг 3: Получить приватный ключ и адрес фандера

Это нужно для того, чтобы бот мог размещать ордера от твоего имени.

1. Открой **https://reveal.polymarket.com**
2. Залогинься тем же способом, что и на Polymarket
3. Нажми **Reveal Private Key**
4. Скопируй:
   - **Private Key** — длинная строка типа `0x4c8f...a3b2`
   - **Proxy Wallet Address** (Funder) — адрес типа `0x7Ef3...9d1A`
5. **СОХРАНИ** оба значения в безопасном месте (менеджер паролей)

⚠️ **Никогда не делись приватным ключом. Кто имеет ключ — имеет доступ ко всем средствам.**

---

## Шаг 4: Добавить секреты в GitHub

1. Открой https://github.com/BRKME/Polycashmashine/settings/secrets/actions
2. Нажми **New repository secret** и добавь три секрета:

| Name | Value | Пример |
|------|-------|--------|
| `POLY_PRIVATE_KEY` | Приватный ключ из шага 3 | `0x4c8f...a3b2` |
| `POLY_FUNDER` | Proxy Wallet Address из шага 3 | `0x7Ef3...9d1A` |
| `POLY_SIG_TYPE` | Тип подписи | `1` (email) или `0` (MetaMask) |

---

## Шаг 5: Первый тест (dry run)

1. Открой https://github.com/BRKME/Polycashmashine/actions
2. Выбери **Weather Bot — Trade**
3. Нажми **Run workflow**
4. Параметры:
   - `dry_run` = **true**
   - `max_bet` = 5
   - `daily_budget` = 50
5. Проверь лог — должен показать сигналы и `[DRY_RUN]` трейды

---

## Шаг 6: Первый реальный трейд (осторожно)

1. **Weather Bot — Trade** → Run workflow
2. Параметры:
   - `dry_run` = **false**
   - `max_bet` = **2** (начни с малого!)
   - `daily_budget` = **20**
3. Проверь лог — должен показать `FILLED` вместо `DRY_RUN`
4. Проверь баланс на polymarket.com — должны появиться позиции

---

## Шаг 7: Масштабирование

Когда убедишься что ордера исполняются корректно:

| Уровень | max_bet | daily_budget | Примечание |
|---------|---------|-------------|------------|
| Тест | $2 | $20 | Первые 3 дня |
| Нормальный | $5 | $50 | После проверки |
| Агрессивный | $10 | $100 | Только при стабильном профите |

Бот запускается автоматически каждые 6 часов (00:30, 06:30, 12:30, 18:30 UTC).

## FAQ

**Сколько нужно денег для старта?**
Минимум $50 USDC. Рекомендую $100 — хватит на неделю торговли с буфером.

**Это безопасно?**
Приватный ключ хранится в GitHub Secrets — зашифрован, не виден в логах. Бот размещает только market orders (Fill-or-Kill) — если цена сдвинулась, ордер отменяется.

**Что если бот ошибётся?**
Максимальный риск = daily_budget ($50). Бот не может потерять больше этой суммы за день.

**Нужен VPN?**
Polymarket заблокирован для US-резидентов. Если ты не в США — VPN не нужен. Ты в Литве — всё OK.

**Как проверить результаты?**
polymarket.com → Portfolio → увидишь все открытые позиции и P&L.
